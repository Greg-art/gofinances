# gofinances

gofinances é um aplicativo para organizar suas finanças, dividindo suas entradas e despesas. 

## Desenvolvimento

Esse aplicativo foi feito durante o curso Ignite(React Native) da Rocketseat, onde aprendi na prática a fazer um aplicativo do zero.

### Tecnologias
O aplicativo foi feito com React-native usando o **Expo** como ferramenta base para auxiliar o desenvolvimento e **Styled Components** para estilização. Possui 
autênticação de login social com conta Google ou conta Apple e usa **Async Storage** para salvar as informações do usuário localmente. Foi utilizado a **context API** 
para manter a autenticação e **React-hook-form** para auxiliar nos formulários.
